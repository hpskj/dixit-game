# بوتات ذكية بالذكاء الصناعي (اختياري)

هذه النسخة تضيف دعم AI Vision للبوتات.

## المطلوب في Render
أضف Environment Variable:

```env
OPENAI_API_KEY=ضع مفتاح OpenAI هنا
```

اختياري للصور المحلية مثل /cards/card1.svg:

```env
AI_IMAGE_BASE_URL=https://www.dixitq8.com
```

اختياري لتغيير النموذج:

```env
OPENAI_MODEL=gpt-4.1-mini
```

## ملاحظات مهمة
- إذا لم تضف OPENAI_API_KEY، البوتات تعمل بالنظام القديم بدون أن يتعطل الموقع.
- إذا فشل طلب الذكاء الصناعي لأي سبب، يرجع البوت تلقائياً للنظام العادي.
- البوت الذكي يستخدم الصور لفهمها ثم يختار الكرت ويكتب التلميح ويصوّت.
- قد تكون هناك تكلفة API حسب الاستخدام وعدد البوتات والجولات.
