# تحديث صورة غلاف الغرفة

## الجديد
- إضافة صورة غلاف مخصصة لكل غرفة.
- من لوحة الأدمن، داخل صور الغرفة، اضغط: ⭐ تعيين كغلاف.
- يظهر الغلاف المختار في قائمة الغرف للاعبين.
- يمكن إزالة الغلاف المخصص، وحينها يعود الموقع لاستخدام أول صورة في الغرفة كغلاف.
- إذا حذفت صورة كانت هي الغلاف، يتم مسح الغلاف تلقائياً حتى لا يظهر رابط صورة محذوفة.

## خطوة Supabase المطلوبة مرة واحدة
شغّل الملف:
SUPABASE_ADD_ROOM_COVER_SQL.sql

أو شغّل هذا السطر فقط في SQL Editor:
alter table rooms add column if not exists cover_image text;
