تحديث حذف الصور الجماعي من لوحة التحكم

ما تمت إضافته:
- مربع تحديد لكل صورة داخل كل غرفة في لوحة الأدمن.
- زر تحديد الكل.
- زر إلغاء التحديد.
- زر حذف الصور المحددة دفعة واحدة.
- API جديد في السيرفر:
  POST /api/admin/room-templates/:roomId/cards/bulk-delete

ملاحظات:
- الحذف الجماعي يزيل الصور من الغرفة فقط، مثل زر "إزالة من الغرفة" السابق.
- تخزين الغرف ما زال في Supabase، لذلك التعديل دائم بعد Redeploy.
- لا تحتاج لتشغيل SQL جديد لهذا التحديث.
