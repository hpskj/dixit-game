# طريقة استخدام غرف الصور في لوحة الأدمن

1. افتح /admin.html وسجل دخول الأدمن.
2. من قسم "إنشاء غرفة / فئة صور":
   - اكتب اسم الغرفة.
   - اكتب فئة الصور.
   - اكتب وصف اختياري.
   - اضغط "إنشاء الغرفة".
3. ستظهر الغرفة في قسم "الغرف الحالية".
4. داخل بطاقة الغرفة اضغط "Choose Files" واختر عدة صور.
5. اضغط "رفع صور لهذه الغرفة".
6. بعد الرفع ستظهر الصور داخل نفس الغرفة.
7. اللاعب بعد تسجيل الدخول يرى الغرف ويختار غرفة، واللعبة تستخدم صور هذه الغرفة فقط.

ملاحظات مهمة:
- يجب ضبط Cloudinary في Render حتى تبقى الصور بعد إعادة تشغيل السيرفر.
- يجب أن يكون SUPABASE_KEY هو service_role key لأن RLS مفعّل.
- إذا لم تظهر التعديلات بعد الرفع إلى Render، اضغط Manual Deploy ثم Clear browser cache.
