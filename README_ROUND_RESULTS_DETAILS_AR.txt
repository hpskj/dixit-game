تحديث عرض نتائج الجولة
======================

الجديد:
- عند انتهاء الجولة تظهر النتائج بشكل تفصيلي.
- كل كرت يظهر في صف مستقل.
- بجانب كل كرت يظهر صاحب الكرت.
- تظهر أسماء اللاعبين الذين اختاروا هذا الكرت.
- تظهر نقاط صاحب الكرت في هذه الجولة.
- يظهر ملخص نقاط الجولة لكل اللاعبين أعلى النتائج.

لا يحتاج هذا التحديث إلى SQL جديد.
ارفع كل الملفات واعمل Redeploy.
