# تحديث منع الوميض + منع تكرار تلميحات البوت

## الجديد
- عند اختيار اللاعب لكرت أو التصويت، لا يتم إعادة رسم الشاشة عند كل اللاعبين.
- يتم تحديث واجهة اللاعب نفسه فقط برسالة/تحديد محلي.
- يتم إرسال تحديث للجميع فقط عند الانتقال للمرحلة التالية أو ظهور النتائج.
- منع تكرار تلميحات البوت داخل نفس المباراة.
- AI Bot يتم منعه من استخدام تلميح سبق استخدامه في نفس المباراة.
- البوت العادي يختار تلميحات عربية فقط وغير مكررة قدر الإمكان.

## لا يحتاج SQL جديد
ارفع كل الملفات واعمل Redeploy فقط.
