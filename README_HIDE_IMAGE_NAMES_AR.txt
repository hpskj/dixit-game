تحديث إخفاء أسماء الصور

تم إخفاء اسم/عنوان الصورة من كروت اللعب ومن كروت الطاولة.

المحافظة على الميزات السابقة:
- زر اللعبة
- إخفاء زر الدعوة العلوي
- مؤقت قرب الأوراق
- مؤقت التصويت 45 ثانية
- طرد لاعب لصاحب الغرفة
- الرجوع للعبة بعد الانقطاع
- صورة غلاف للغرفة
- الحذف الجماعي
- Supabase للغرف

لا يحتاج SQL جديد.
