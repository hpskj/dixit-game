ميزة طرد لاعب بواسطة صاحب الغرفة
================================

الجديد:
- يظهر زر "طرد" بجانب كل لاعب لصاحب الغرفة فقط.
- صاحب الغرفة لا يستطيع طرد نفسه.
- اللاعب المطرود يرجع لقائمة الغرف وتظهر له رسالة.
- اللاعب المطرود لا يستطيع الرجوع لنفس المباراة باستخدام زر الرجوع أو كود الغرفة.
- إذا تم طرد لاعب أثناء الجولة، يتم تنظيف كروته وتصويته واختياره من الجولة الحالية.
- إذا تم طرد الراوي أثناء جولة نشطة، تبدأ جولة جديدة تلقائياً حتى لا تتعلق المباراة.

لا يحتاج SQL جديد.

طريقة التحديث:
1) ارفع كل ملفات هذه النسخة فوق النسخة الحالية.
2) اعمل Redeploy في Render.
3) اختبر بصاحب الغرفة ولاعب آخر.
