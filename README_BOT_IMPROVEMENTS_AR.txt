تحسين البوتات
================

تم تحسين البوتات بدون الحاجة إلى SQL جديد أو API خارجي.

الجديد:
- مستويات ذكاء مختلفة للبوت: سهل / عادي / ذكي.
- البوت عندما يكون الراوي يختار كرتًا أنسب للتلميح بدل العشوائية الكاملة.
- تلميحات البوت أصبحت أكثر تنوعًا وأقرب لعنوان/اسم الصورة إن وجد.
- أثناء اختيار كرت للمشاركة، البوت يحاول اختيار كرت قريب من تلميح الراوي.
- أثناء التصويت، البوت يحاول ربط التلميح بالكروت، لكن مع نسبة خطأ حتى تبقى اللعبة ممتعة.
- لا يحتاج إعدادات جديدة ولا مفاتيح OpenAI.

ملاحظة:
هذا تحسين ذكي محلي. إذا أردت لاحقًا بوت يفهم الصورة نفسها فعليًا، نحتاج ربط API ذكاء اصطناعي بصري مثل OpenAI/Gemini.
