تحديث حذف جماعي من قسم كل الصور العامة

الجديد:
- إضافة مربع تحديد لكل صورة في قسم "كل الصور العامة".
- زر تحديد الكل.
- زر إلغاء التحديد.
- زر حذف المحدد نهائياً.

ملاحظات:
- الحذف النهائي يحذف الصورة من النظام.
- إذا كانت الصورة مضافة داخل أي غرفة، سيتم إزالتها من الغرف أيضاً حتى لا تظهر روابط مكسورة.
- لا تحتاج إلى SQL جديد.

طريقة التحديث:
1. ارفع هذه النسخة إلى GitHub أو Render حسب طريقتك.
2. اعمل Redeploy.
3. افتح لوحة التحكم وجرب تحديد عدة صور من قسم "كل الصور العامة" ثم حذفها.
