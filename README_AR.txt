لعبة خيال أونلاين - النسخة V6

الإضافات الجديدة:
1) مؤقت 30 ثانية لاختيار الراوي/الكروت.
2) مؤقت 20 ثانية للتصويت.
3) لوحة تحكم لتغيير وقت المؤقتات.
4) أصوات عند اختيار الكرت وعند الفوز.
5) تحسين كبير للتصميم والشكل.
6) استمرار لوحة التحكم لإضافة صورة واحدة أو مجموعة صور دفعة واحدة.

طريقة التشغيل على Windows:
1) فك الضغط عن الملف.
2) افتح Terminal أو CMD داخل مجلد اللعبة.
3) اكتب:
   npm install
4) ثم اكتب:
   npm start
5) افتح المتصفح:
   http://localhost:3000

لوحة التحكم:
http://localhost:3000/admin.html

مهم:
- لا تغلق شاشة السيرفر السوداء أثناء اللعب.
- تغيير المؤقتات من لوحة التحكم يطبق عند بدء لعبة جديدة.
- إذا انتهى وقت الاختيار أو التصويت، السيرفر يختار تلقائيًا حتى لا تتوقف الجولة.


تحديث V6 - حماية لوحة التحكم:
- لوحة التحكم الآن خاصة بالأدمن فقط.
- رابط اللوحة:
  http://localhost:3000/admin.html
- عند الدخول سيطلب كلمة مرور.
- كلمة المرور الافتراضية للتجربة:
  123456

مهم جدًا قبل النشر أونلاين:
غيّر كلمة مرور الأدمن. على Render أو أي استضافة أضف Environment Variable باسم:
ADMIN_PASSWORD
واكتب كلمة مرور قوية خاصة فيك.

اختياري لزيادة الأمان:
أضف أيضًا Environment Variable باسم:
ADMIN_TOKEN
واكتب قيمة طويلة وعشوائية.

الآن اللاعبون العاديون لا يستطيعون فتح لوحة التحكم أو رفع/حذف الصور أو تغيير المؤقتات.


تحديث V7:
- عند نهاية كل جولة تظهر كل الكروت مع اسم صاحب الكرت.
- تظهر أسماء اللاعبين الذين صوّتوا/اختاروا كل صورة.
- يتم تمييز كرت الراوي في شاشة النتائج.

تحديث V9 - حماية لوحة التحكم:
- تم حذف زر لوحة التحكم من صفحة اللعبة نهائيًا.
- لوحة التحكم لا تفتح إلا بعد تسجيل الدخول بكلمة مرور.
- رابط لوحة التحكم المباشر: http://localhost:3000/admin.html
- صفحة الدخول: http://localhost:3000/admin-login.html
- كلمة المرور الافتراضية محليًا: 123456
- قبل النشر أونلاين غيّر كلمة المرور من متغير البيئة:
  ADMIN_PASSWORD=كلمة_مرور_قوية
- لا تشارك كلمة المرور مع اللاعبين.


ملاحظة V10 - رفع الصور الاحترافي:
هذه النسخة تدعم Cloudinary لحفظ الصور بشكل دائم على Render.
أضف هذه المتغيرات في Render > Environment:
CLOUDINARY_CLOUD_NAME
CLOUDINARY_API_KEY
CLOUDINARY_API_SECRET
ثم اختر Save, rebuild, and deploy.

بعد رفع الصور من لوحة التحكم ستُحفظ في Cloudinary داخل مجلد dixitq8/cards، وستظهر دائمًا حتى بعد إعادة تشغيل Render.
