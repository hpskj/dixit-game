# ميزة الرجوع للعبة بعد الخروج أو انقطاع الاتصال

تمت إضافة زر يظهر للاعب بعد تسجيل الدخول باسم: رجوع للعبة السابقة.

## كيف تعمل؟
- عند إنشاء أو دخول مباراة، يتم حفظ كود آخر لعبة في متصفح اللاعب.
- السيرفر يبحث عن أي مباراة نشطة لنفس حساب اللاعب.
- عند الضغط على زر الرجوع، يتم استخدام reconnectRoom لإرجاع اللاعب بنفس النقاط والكروت والمرحلة.

## ملاحظات مهمة
- تعمل طالما المباراة ما زالت موجودة في ذاكرة السيرفر.
- إذا أعيد تشغيل السيرفر بالكامل، المباريات الجارية نفسها قد تنتهي لأنها غير محفوظة في قاعدة البيانات.
- الغرف/الفئات محفوظة في Supabase ولا تتأثر.
